# Fashion-Avatar-Project

**All UI development should happen inside the `src/` directory. For example:** 

*  `src/wardrobe/` — (Assigned to: [Member 1])
*  `src/avatar/` — (Assigned to: [Member 2])
*  `src/ai-textbot/` — (Assigned to: [Member 3])

The main dashboard layout will be hardcoded in `src/app/page.tsx`. It acts as a shell that imports our stuff in src folders.

I also downloaded some dependencies, so there could be a lot of files.

* Programming Language: Typescript
* UI Library: React
* Styling: Tailwind CSS
* Framework: Next.js

To run the code, go to root folder '/fashion-avatar-project', type 'cmd' in the Address bar of file explorer, and type 'npm run dev'. It should give you a link to localhost that you can click on.
If you use a Macbook, I think there should be a 'New Terminal at Folder' button if you right-click on the folder. After that, you can type 'npm run dev' in the terminal.
